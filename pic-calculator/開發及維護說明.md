# 圖片計算器 - 開發及維護說明

## 專案概述

圖片計算器是一個基於網頁的表格辨識與驗算工具，支援：
- 圖片上傳 OCR 辨識表格
- 手機拍照即時辨識
- 橫向/縱向驗算
- PWA 可安裝為手機 App

## 技術架構

```
pic-calculator/
├── app/                   # 後端 API
│   ├── main.py           # FastAPI 伺服器
│   ├── table_ocr.py      # PaddleOCR 表格辨識
│   └── verifier.py       # 驗算邏輯（已移至前端）
├── static/               # 前端靜態檔案
│   ├── index.html        # 主頁面（含完整前端邏輯）
```

## 功能說明

### 1. 圖片上傳與辨識

- 支援拖曳或點擊選擇圖片
- 支援 JPG/PNG/WEBP 格式
- 自動偵測表格邊界並解析

### 2. 驗算功能

- 點擊**欄號**（頂部數字）→ 進行欄驗算
- 點擊**列號**（左側數字）→ 進行列驗算
- 點擊循環：+（加項）→ −（減項）→ 取消
- 最後選取的欄/列自動作為合計（=）

### 3. 手機相機

- 開啟相機進行即時拍照辨識
- 需使用 HTTPS 連線（行動裝置限制）

### 4. PWA 安裝

- iOS：Safari → 分享 → 加入主畫面
- Android：Chrome → 選單 → 安裝應用程式

## 驗算邏輯（前端）

位於 `static/index.html` 中的 `Validator` 物件：

```javascript
// 橫向驗算（欄）
case 'horizontal':
  inputs.forEach(i => sum += value * sign);

// 縱向驗算（列）
case 'vertical_row':
  inputs.forEach(r => sum += value * sign);
```

容差計算：差異不超過 `max(1.0, |實際值| * 0.005)` 視為正確。

## 維護指南

### 新增功能

1. **前端**：修改 `static/index.html`
2. **後端 API**：修改 `app/main.py`
3. **OCR 邏輯**：修改 `app/table_ocr.py`

### 部署至手機

#### 方法一：本地網路（HTTP）

1. 確認防火牆允許 port 8000
2. 手機連線同 WiFi，輸入 `http://<電腦IP>:8000`

#### 方法二：HTTPS（相機功能必備）

```bash
# 安裝 ngrok
brew install ngrok
ngrok http 8000
```

用 ngroy 提供的 HTTPS 網址訪問。
